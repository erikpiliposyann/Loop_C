# Logic and Numbers - C Programming Exercises

This folder contains C programming exercises focused on number processing, mathematical logic, and algorithmic thinking.

The main goal of these tasks is to improve problem-solving skills by working with digits, sequences, recursion-like logic, and prime-based computations.

---

## 📌 Topics Covered

- Number manipulation (digits, reversal, checks)
- Mathematical logic
- Loops and conditional statements
- Algorithmic problem solving
- Basic number theory (prime numbers, Fibonacci sequence)

---

## 📂 Exercises Included

### 11. Print digits of a number
Extracts and prints digits of a given number in correct order.

### 12. Palindrome check
Checks whether a number is a palindrome (reads the same forwards and backwards).

### 13. Fibonacci number
Finds the Fibonacci number at a given index using iterative approach.

### 14. Prime number check
Determines whether a number is prime by checking divisibility up to √n.

---

## 🚀 Purpose

These exercises help to develop:

- Strong logical thinking
- Understanding of mathematical algorithms
- Working with integer operations
- Efficient loop design

---

## 🧑‍💻 Skills Practiced

- Mathematical reasoning in C

After completing these tasks, you should be comfortable with:

- Implementing classic algorithms (Fibonacci, prime check)
