# Patterns - C Programming Exercises

This folder contains C programming exercises focused on visual output (patterns) and nested loop logic.

These tasks help improve understanding of loops, indexing, and structured output formatting in C.

---

## 📌 Topics Covered

- Nested loops (`for`, `while`)
- Console pattern generation
- Mathematical patterns
- Digit-based logic
- Sequence analysis (arithmetic & geometric progression)
- Visual problem solving

---

## 📂 Exercises Included

### 1. Multiplication Tables (1–10)
Prints full multiplication tables from 1 to 10.

---

### 2. Number Triangle (Repetition Pattern)
Prints a triangle where each row contains repeated numbers.

---

### 3. Square Pattern (*)
Prints a filled square using `*` characters.

---

### 4. Sequential Number Triangle
Prints increasing numbers in triangular form.

---

### 5. Floyd’s Triangle
Prints Floyd’s triangle using 0 and 1 pattern logic.

---

### 6. Hollow Square
Prints a square with only borders filled with `*`.

---

### 7. Triangle Shape
Prints a triangle structure based on input height.

---

### 8. Arithmetic Progression Check (Digits)
Checks whether the digits of a number form an arithmetic progression.

**Logic:**  
- Extract digits
- Compute common difference
- Verify consistency across all digit pairs

Output:
- `True` → digits form arithmetic progression  
- `False` → otherwise

---

### 9. Geometric Progression Check (Digits)
Checks whether the digits of a number form a geometric progression.

**Logic:**  
- Compare ratios between consecutive digits
- Verify constant multiplication relationship

Output:
- `True` → digits form geometric progression  
- `False` → otherwise
