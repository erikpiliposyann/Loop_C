# Basics - C Programming Exercises

This folder contains basic C programming exercises focused on fundamental concepts such as input/output, conditional statements, loops, and simple arithmetic operations.

These tasks are designed for beginners to strengthen core programming skills and logical thinking in C.

---

## 📌 Topics Covered

- Input and Output (`scanf`, `printf`)
- Conditional statements (`if`, `else`, `switch`)
- Loops (`for`, `while`)
- Basic arithmetic operations
- Simple problem solving logic

---

## 📂 Exercises Included

### 1. Print numbers (0–100 except specific values)
Prints numbers from 0 to 100, excluding 3, 5, and 15.

### 2. Sum from 1 to N
Calculates the sum of all integers from 1 to a given number N.

### 3. Sum of odd numbers
Calculates the sum of all odd numbers from 1 to N.

### 4. Number guessing game
User tries to guess a secret number with hints (“higher” / “lower”).

### 5. Multiplication table (single number)
Prints multiplication table for a given number.

### 7. Multiplication without using *
Multiplies two numbers using only addition and loops.

### 8. Power calculation
Computes `base^exponent` using repeated multiplication.

### 9. Sum of positive numbers until 0
Continuously reads numbers and sums only positive ones until 0 is entered.

### 10. Vowel or consonant checker
Checks whether a given character is a vowel or consonant using `switch`.

The goal of these exercises is to build a strong foundation in:
